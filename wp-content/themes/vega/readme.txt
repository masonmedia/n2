== Description ==

Vega is a clean, one page business WordPress theme. It has a custom frontpage template to showcase special timeless elements/information. Vega features:

* Three pre-built color schemes that can be switched/chosen from the Customizer
* Three blog feed views (Excerpt Only, Small Image on Left Excerpt on Right, Large Image and Full Content)
* Multiple options for banners
* Ability to turn sidebar on or off on pages and posts
* Custom front page template that allows for multiple sections such as two call to action rows, featured pages, recent blog posts, and open content
* Call to Action rows have a parallax effect
* Front page banner has parallax effect
* Note: The site tagline appears on the home/front page only on top of the header image/banner

For a complete list of features and detailed documentation, please go to https://www.lyrathemes.com/vega/
